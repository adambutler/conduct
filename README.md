# Conduct

Conduct.io is a Ruby-on-rails application for running instant hacker-news style sites. Create a topic and then collect responses easily.

## Installing

```
$ git clone git@github.com:adambutler/conduct.git
$ cd conduct
$ bundle install
$ rake db:create
$ rake db:migrate
```
